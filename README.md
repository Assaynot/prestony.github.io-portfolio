# Portfolio BTS SIO — option SISR

Portfolio statique multi-pages pour l'épreuve E4, conforme aux sections obligatoires
du référentiel (présentation, BTS SIO, projets/PPE, veille, patrimoine informatique,
synthèse des compétences, contact) + une page **Synoptique E5** reprenant la trame
de `Trame_synoptique_PortFolio.xlsx`.

## Structure
- `index.html` … accueil
- `about.html` … profil & parcours
- `bts-sio.html` … présentation du diplôme et de l'option SISR
- `projets.html` … PPE / ateliers (AD, GLPI, Cyber, Cisco)
- `veille.html` … veille technologique & juridique
- `patrimoine.html` … gestion du patrimoine informatique
- `competences.html` … tableau de synthèse des compétences
- `synoptique.html` … trame de l'oral E5
- `contact.html` … formulaire de contact
- `css/style.css`, `js/main.js`, `cv.pdf`

## À personnaliser
Remplacez tous les marqueurs `[ ... ]` (nom, liens LinkedIn/GitHub, etc.) et les
encadrés ambre « À compléter ». Ajoutez vos captures dans les emplacements prévus
et remplacez `cv.pdf`.

## Mise en ligne (gratuit)
Poussez le dossier sur un dépôt GitHub puis activez **GitHub Pages**
(Settings → Pages → branche `main`, dossier `/root`).
